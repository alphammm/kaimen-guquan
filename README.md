# 开门古泉 — 中国古钱币收藏数据库

> Chinese Ancient Coin Collection Database

一个全面的中国古钱币交互式数据库网站，涵盖从先秦到民国的所有朝代，古风设计，包含 AI 鉴宝功能。

## 预览

直接用浏览器打开 `index.html` 即可使用。

## 特色功能

- **全朝代覆盖** — 先秦贝币/刀布币 → 秦半两 → 汉五铢 → 唐开元 → 宋崇宁 → 明清通宝 → 机制币/民国币，100+ 代表性钱币
- **朝代导航** — 横向滑动的朝代时间线，一键筛选
- **拍卖成交记录** — 历史拍卖数据，可排序可搜索
- **AI 鉴宝** — 模拟 AI 古币鉴定分析功能
- **古风设计** — 朱红/金色/墨黑配色 + 书法字体 + 水墨渐变 + 回纹边框

## 技术栈

- 纯前端：HTML + CSS + JavaScript
- 无框架依赖，零构建步骤
- Google Fonts: Ma Shan Zheng (书法) + Noto Serif SC (宋体)

## 文件结构

```
├── index.html      # 主页面
├── styles.css      # 古风设计系统样式
├── data.js         # 全量古钱币数据库（朝代 + 钱币 + 拍卖记录）
├── app.js          # 交互逻辑（搜索/筛选/动画/AI鉴宝）
└── README.md
```
